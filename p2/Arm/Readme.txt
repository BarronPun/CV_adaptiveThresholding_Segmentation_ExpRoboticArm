A150 folder contatins the files to use the robotics arm in the lab.
The step-by-step instructions are provided in the script name "Example_Script_Instruction_Howto". Please open it in MATLAB and set you MATLAB current folder pointing to the A150 sitting next to this text file.

Be aware that before running the script, you need to put the "main power" switch to on mode. 

Then by holding the arm in according to the alignment signs on each joint, press the orange "arm power".

Now you can run the MATLAB script. 

Further instructions are provided in the script.
